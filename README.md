# FabFour Foundation App

A comprehensive web application for the FabFour Foundation, designed to connect students, donors, and counselors in an educational support ecosystem.

## 🎯 Project Overview

The FabFour Foundation App is a multi-module platform that enables:
- **Students**: Create campaigns for academic needs, manage profiles, and track funding
- **Donors**: Browse campaigns and make anonymous or registered donations
- **Counselors**: Verify student applications and provide guidance
- **Admins**: Manage the platform and oversee operations


## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd fabfour-app
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```
